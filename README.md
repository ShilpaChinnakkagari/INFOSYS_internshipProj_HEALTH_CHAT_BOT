# Health Wellness Chatbot

An intelligent digital health assistant designed to promote wellness and provide preventive healthcare guidance.

## Features

### User Features
- **AI Health Chatbot**: Interactive health advice and guidance
- **Health Score Tracking**: Monitor wellness progress with visual graphs
- **Profile Management**: Update personal information and preferences
- **Emergency Assistance**: Instant connection to emergency services
- **Multi-language Support**: Accessible in multiple languages

### Admin Features
- **User Management**: View all registered users
- **System Analytics**: Access user activity reports
- **Content Management**: Add/update health tips and precautions
- **Dashboard**: Overall system statistics and insights
